# P2P Chat

A peer to peer chat application which is end to end encrypted and does not require login to ensure complete user privacy

## Requirements
- Rust toolchain
- Deno

# Steps:
```bash
git clone https://github.com/mav3ri3k/p2p-chat.git
cd p2p-chat
deno install

# For Desktop development, run:
  deno task tauri dev
```
